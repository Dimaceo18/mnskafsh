# Афиша Минска — Telegram бот

Автоматический сбор и публикация событий в Telegram-канал.

## Источники
- afisha.relax.by
- ticketpro.by
- bycard.by
- 24afisha.by
- family.by

## Функции
- 🔄 Автоматический парсинг каждые 4 часа
- 📢 Публикация новых событий в канал
- 🌅 Утренний дайджест в 09:00
- 📊 Команды для админа: `/parse`, `/stats`, `/settings`

## Команды бота
| Команда | Описание |
|---------|----------|
| `/start` | Главное меню |
| `/parse` | Запустить парсинг (только админ) |
| `/stats` | Статистика публикаций (только админ) |
| `/settings` | Текущие настройки (только админ) |

## Деплой на Render
1. Форкните репозиторий
2. Добавьте переменные окружения в Render:
   - `BOT_TOKEN`
   - `CHANNEL_ID`
   - `ADMIN_ID`
3. Нажмите "Deploy"
